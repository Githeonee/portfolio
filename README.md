# 외국인 유학생 주거 불안정 데이터 분석 포트폴리오

> KCI 등재 논문 기반 실증 데이터 분석 프로젝트
> 정헌 · 최운덕 · 이천재 (강원대학교 부동산학과)  
> 『住居環境』 제23권 제4호, pp.129–151 | [DOI: 10.22313/reik.2025.23.4.129](https://doi.org/10.22313/reik.2025.23.4.129)

---

##  프로젝트 개요

한국의 비수도권 중소도시는 급속한 고령화와 저출산으로 심각한 인구구조 변화를 겪고 있다. 이 위기에 대응하기 위한 핵심 전략 중 하나가 외국인 유학생 유치이지만, 현행 정책은 인구 유입 장려에만 편중되어 있고 실질적인 주거 지원 정책은 미비한 상황이다.

이 프로젝트는 강원대학교·한림대학교 재학 외국인 유학생 69명의 설문 응답 원시 데이터를 직접 수집·분석하여, 춘천시 외국인 유학생이 겪는 주거 불안정의 구조적 요인을 규명하고 정책적 대안을 탐색한다.

## 핵심 연구 질문

| # | 연구 질문 |
|---|-----------|
| RQ1 | 춘천시 내 외국인 유학생의 주거 실태와 만족도는 어떠한가? |
| RQ2 | 이들이 주거 과정에서 겪는 주요 어려움과 그 원인은 무엇인가? |
| RQ3 | 유학생의 주거 안정을 위한 지역 협력형 공공 플랫폼은 어떻게 설계할 수 있는가? |

---

##  주요 발견 (Key Insights)

### 1. 기숙사 거주자가 오히려 더 불만족한다
> 기숙사 외 거주자 만족도 M=3.51 vs 기숙사 거주자 M=2.67 (t=-2.64, p=.012)

직관에 반하는 결과다. 기숙사 거주 유학생의 불만족(1~2점) 비율은 45.8%로, 기숙사 외 거주자(15.6%)의 약 3배에 달했다. 원인은 공용시설 부족, 소음, 시설 노후화로 확인되었다.

### 2. 주거비가 높다고 만족도가 높아지지 않는다
> Spearman ρ = -0.118 (p=.335, 유의하지 않음)

비용과 만족도 사이에 통계적으로 유의한 상관관계가 없다. 단순 주거비 지원보다 정보 접근성·공간 구성·계약 안전성이 만족도에 더 결정적으로 작용함을 시사한다.

### 3. 유학생의 87%가 주거 지원 정책을 모른다
> 정책 인지율: 전체 13.0% (기숙사 외 거주자 기준 6.7%)

정책이 없는 것이 아니라 정보 전달 체계 자체가 작동하지 않는다. 유학생 대부분은 공적 채널이 아닌 지인 추천(29.0%)과 비공식 직거래에 의존한다.

### 4. 주거 탐색의 주요 경로는 비공식 네트워크다
> 학교·기숙사 배정 30.4% / 지인 추천 29.0% / 온라인 플랫폼 17.4% / 부동산 중개 13.0%

공인중개사를 통한 정식 계약 비율은 13%에 불과하며, 중개 수수료를 피하기 위한 직거래 관행이 보증금 미반환, 계약 조건 오해 등의 분쟁으로 이어지고 있다.

## 데이터 개요

| 항목 | 내용 |
|------|------|
| 수집 기간 | 2025년 4월 30일 ~ 6월 23일 (약 8주) |
| 수집 방법 | Google Forms 온라인 설문 (한국어·영어·중국어 3개 언어) |
| 표본 크기 | N = 69 (춘천시 소재 대학 전체 유학생 약 2,200명의 약 3%) |
| 대상 | D-2(유학) 체류자격 보유 외국인 유학생 |
| 국적 분포 | 15개국 이상 (중국, 우즈베키스탄, 네팔, 일본, 인도네시아 등) |
| 조사 대학 | 강원대학교, 한림대학교 |

### 변수 구성

```
설문 구성 (14개 변수)
├── 인구사회학적 특성
│   ├── 성별
│   ├── 연령대
│   └── 국적 및 체류자격
├── 주거 실태
│   ├── 주거 형태 (기숙사/기숙사 외)
│   ├── 월 평균 주거비용
│   └── 주거지 탐색 방법
├── 주거 만족도 (5점 Likert 척도)
└── 주거 과정의 어려움
    ├── 언어 장벽 및 정보 부족 경험
    ├── 계약 과정 불안 요인
    ├── 초기 탐색 시 주요 어려움
    ├── 주거 지원 정책 인지도
    └── 계약 종료 후 분쟁 경험
```

## 분석 방법

| 분석 | 방법 | 적용 변수 |
|------|------|-----------|
| 기술통계 | 빈도분석, 평균, 표준편차 | 전 변수 |
| 집단 간 차이 | 독립표본 t-검정 (Welch) | 주거형태 × 만족도 |
| 범주형 연관성 | 교차분석 (χ²), Fisher 정확검정 | 주거형태 × 만족도그룹, 주거형태 × 정책인지도 |
| 상관분석 | Spearman 순위상관 | 주거비용 × 만족도 |
| 질적 분석 | 주제별 코딩 | 개방형 응답 |
